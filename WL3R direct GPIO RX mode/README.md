# Discover the STM32WL3R workshop for low-power remote control designs

Throughout the https://content.st.com/stm32wl3r-online-workshop.html session, you explored ST’s comprehensive hardware, RF, and software, using the STM32WWL3R, its Nucleo development board, and the STM32Cube ecosystem.

You have learned during specific hands-on session how to implement Direct Gpio mode radio application using OOK modulation

You will find in this repository the code example based on STM32CubeWL3 package that is implementing this direct Gpio Rx mode using Nucleo-WL3RKB2 board

# The material used during session

* the PDF slide used during this hands-on session is available on main repository and explaining complete setup for communication setup and how configure Hw and Sw



# how to replicate

To replicate the direct Gpio communication hands-on you will need

* [NUCLEO-WL3RKB2](https://www.st.com/en/evaluation-tools/nucleo-wl3rkb2.html)
* [STM32CubeIDE](https://www.st.com/en/development-tools/stm32cubeide.html)
* The FW used to generate this application, in the zip file of this folder (unzip and project path : \\Projects\\NUCLEO-WL3RKB2\\Demonstrations\\MRSUBG\\MRSUBG\_Rx\_Data\_DirectGpio\\STM32CubeIDE)
* [STM32Cube Wise Radio Explorer](https://www.st.com/en/development-tools/stm32cubewisere.html) PC App installed on your laptop
